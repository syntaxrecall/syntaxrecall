CSV stands for comma-separated values, a delimited text file format that uses a comma to separate values. Each line of the file is a data record.

# Syntax

```
name,id,favorite food
quincy,1,hot dogs
beau,2,cereal
abbey,3,pizza
mrugesh,4,ice cream
```
