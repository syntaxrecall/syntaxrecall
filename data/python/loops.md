## For loop
```python
array = ['Foo', 'Bar']

for item in array:
  print(item)
```

Output
```
Foo
Bar
```

## For loop with index
```python
array = ['Foo', 'Bar']

for i in range(len(array)):
  print(i, array[i])
```

Output
```
0 Foo
1 Bar
```
